# Finite_Automata
This repository contains C++ libraries for working with: \
      - Finite State Machines aka Finite Automata \
      - Regular Expressions (TODO)

Quick Start \
      1) Run build.sh script \
            $ chmod +x build.sh \
            $ ./build.sh \
      2) Go to bin directory \
            $ cd bin \
      3) To run tests: \
            $ ./AutoTest \
      4) To run main executable file: \
            $ ./Auto.out
